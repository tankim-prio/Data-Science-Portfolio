## Featured Engineering Projects

### 01. [Computational Foundations](./01_Python_Fundamentals)
* **Objective:** Implementing efficient data structures and modular logic.
* **Key Implementations:** Mastery of Pythonic structures (Lists, Dicts, Tuples, Sets) to optimize data retrieval and custom parametric functions.
* **Significance:** Establishes the core programming discipline required for scalable software and AI development.

### 02. [Statistical Data Wrangling & EDA](./02_Data_Analysis_and_EDA)
* **The Problem:** Raw student data often contains noise and missing values that threaten the integrity of statistical analysis.
* **The Solution:** Engineered a robust cleaning pipeline using **Mean/Median imputation** and NumPy-based statistical masking.
* **Key Insights:** Identified socioeconomic correlations (e.g., impact of lunch types and test prep) on academic outcomes via complex `.groupby()` aggregations.

### 03. [Visual Intelligence & Analysis](./03_Data_Visualization)
* **The Problem:** Interpreting high-dimensional consumer behavior requires advanced visual distribution analysis.
* **The Solution:** Developed a visualization suite using Matplotlib to decode fitness industry trends.
* **Key Outputs:** Time-frequency histograms for usage patterns and multi-variable scatter plots to identify user behavior clusters.

### 04. [Predictive Analytics: Real Estate Regression](./04_Predictive_Regression)
* **The Problem:** Property valuation requires a model capable of handling varied feature scales and categorical architectural data.
* **The Solution:** Built a Multiple Linear Regression pipeline featuring **StandardScaler** for normalization and **One-Hot Encoding** for categorical features.
* **Metrics:** Evaluated model generalizability using **RMSE** and **R-squared** metrics to ensure reliability on unseen data.

### 05. [Clinical Diagnostics: Advanced ML Classification](./05_Advanced_Classification)
* **The Problem:** In medical diagnostics, "False Negatives" (missing a malignant tumor) are a critical life-safety risk.
* **The Solution:** Developed a high-precision Decision Tree classifier for breast cancer diagnosis.
* **Optimization:** Leveraged **GridSearchCV** for hyperparameter tuning and **SelectKBest** for statistical feature selection.
* **Performance:** Achieved a **96.4% ROC-AUC score**, demonstrating elite-level diagnostic accuracy.

## Technical Methodology

I adhere to a systematic machine learning pipeline to ensure all models are reproducible and statistically sound:



1.  **Preprocessing:** Robust feature scaling, encoding, and imputation strategies.
2.  **Engineering:** Deriving interaction and polynomial features to maximize model signal.
3.  **Optimization:** Utilizing cross-validation to find the global optimum for model parameters.
4.  **Validation:** Rigorous assessment via Precision, Recall, F1-Score, and AUC-ROC curves.
